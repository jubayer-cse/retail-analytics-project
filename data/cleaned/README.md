### What the cleaning process did
- Removed ~243,000 rows with missing CustomerID
- Removed cancelled orders (InvoiceNo starting with C)
- Removed negative quantities (returns) and zero/negative prices
- Removed duplicate rows
- Created TotalPrice column (Quantity × UnitPrice)
- Converted InvoiceDate to proper datetime format
- **Result: 779,425 clean rows ready for SQL and Power BI**
